HTML

This is an HTML file. HTML is the programming language that tells a web browser how to display information, like text or images. While this whole document is HTML, the only visible parts are inside the <body> element.

We usually don’t just write HTML. In fact, there are three languages for the web: HTML, CSS, and JavaScript.

CSS

CSS lets us design, or “style,” HTML elements. We write CSS in the <head> section of the HTML document, inside of <style> tags.

JavaScript (JS)

JavaScript is the fanciest of all the languages. It lets us write HTML and CSS generatively, as well as make our website more meaningfully interactive. You’ll usually find it right at the end of the <body> element, inside of <script> tags.
