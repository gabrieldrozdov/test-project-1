Make Art > Learn Code

Creative web projects that’ll teach you HTML, CSS, and JavaScript.

Learn code naturally by tackling creative challenges

Whether you’re coding for the first time or trying to pick it back up, it’s easy to hit a wall and feel discouraged. It doesn’t have to be that way! In this website, you’ll find a series of creative projects that will help you gradually pick up coding skills while always making sure you get little victories to keep you motivated.

You’ll also pick up the skills needed to become a self-sufficient coder. Instead of articles or lectures, this website features numerous links to helpful online resources that professional developers use as they code, too. (Big shout-outs to MDN and W3Schools!)

Play with live demos directly in your web browser

Setting up a development environment on your computer is one of the first challenges of coding. You have to install a text editor, get on GitHub, and learn how to launch a website. While that’s a really important part of creating websites, it’s also something that can stop new coders before they even write a line of code.

If you’re still in that phase, good news! This site features a built-in text editor that let’s you play with code right in your browser. And once you’re ready to code on your own, you can download the project demos straight from this website’s GitHub repository.

Websites work on virtually any device, are easy to get started developing for, enable interactive experiences from the jump, and have a vibrant decades-old community of designers and developers dedicated to making them better and better.

If you find these projects inspiring, then don’t stop! Once you learn JavaScript, you can start working with libraries that enable generative art, audio synthesis, physics simulation, 3D modeling, data visualization, machine learning, and so much more. Background

One way in which websites are unique is their ability to take on many forms to fit any screen size. To accomplish this, websites are less like images or books, and more like instructions for creating an image or book. Those instructions are written as code in the form of HTML.

HTML is the backbone of the web. It’s a programming language, which means that computers can read it like a set of instructions to display information on your screen. Unlike traditional languages, programming languages have to follow super precise rules to be understood by computers. We call these rules the language’s syntax. HTML’s syntax is structured around HTML tags, which create HTML elements.

Prompt

In this project, let’s explore the peculiarities of websites and HTML.

Create an HTML file. Using just text, design a website that encourages users to resize the browser window.

Questions to Explore

What happens if you try to press “return” or ”enter” to break to a new line?

What happens if you try to add multiple spaces in a row? (Hint: nothing happens! How can you add multiple spaces in a row?)

How can you italicize or bold text?

Is everything in your HTML file visible on the live website?
